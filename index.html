  1. Download the selected website.
  2. Compile its css to scss(new version of sass) – the reason is scss document is very organized, much easier to read and find what you are looking for than css.
  3. Learn sass, learn kss – very difficult to learn – little to non syntax guide. We learned it by constant trying and testing.
  4. Use kss on scss format to build the style guide website.
  5. Compile css of style guide web site to scss. Polish it. Then add js such as jquerry to make it nicer.
